# Curtis Tinkers CMS Blog

Sets up blog content type.

## Recipes Applied
- [Curtis Tinkers CMS Media](https://github.com/curtistinkers/curtistinkers_cms_media)
- [Drupal CMS Content Basics](https://www.drupal.org/project/drupal_cms_content_type_base)
- [Drupal CMS SEO Tools](https://www.drupal.org/project/drupal_cms_seo_tools)

## Modules and Themes Installed
- Canvas (core)
- CKeditor (core)
- Comment (core)
- Content (core)
- [highlight.js Input Filter](https://www.drupal.org/project/highlightjs_input_filter)
- [Linkit](https://www.drupal.org/project/linkit)
- [MaxLength](https://www.drupal.org/project/maxlength)
- [Scheduler](https://www.drupal.org/project/scheduler)
- [Tagify](https://www.drupal.org/project/tagify)
- [UI Icons](https://www.drupal.org/project/ui_icons)
- [Advanced Views RSS Feed](https://www.drupal.org/project/views_rss)
- Views (core)

## Installation
This project is part of [Drupal CMS](https://drupal.org/project/cms). If you're not using Drupal CMS, you can still use this recipe in your own project. Add it with the following Composer command:

```
composer require drupal/curtistinkers_cms_blog
```
