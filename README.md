## Installation

This project is part of Curtis Tinkers CMS which is based on [Drupal CMS](https://drupal.org/project/cms). If you're not using Curtis Tinkers CMS or Drupal CMS, you can still use this recipe in your own project. Add it with the following Composer command:

```
composer require drupal/curtistinkers_cms_blog
```
